# Projeto.Ransoware.python
Projeto exclusivamente para fins educativos em ambiente controlado, demonstrando a ferramenta de criptografia e descriptografia.

#TRECHO DE UM CÓDIGO DE RANSOWARE. 


Função que percorre recursivamente um diretório e retorna os caminhos completos dos arquivos encontrados, excluindo determinados nomes ou extensões.



![trecho do codigo criptografado encontrando local](https://github.com/user-attachments/assets/d8a01382-a815-49a7-a3fb-0c3fcf353cb9)

#TRECHO DE UM CÓDIGO DE RANSOWARE. 

Esse código carrega uma chave de descriptografia, percorre todos os arquivos dentro do diretório test_files, aplica a função de descriptografia em cada um deles e, ao final, exibe no console a mensagem: Descriptografia concluída! Arquivos restaurados
